Fortran - ¡Hola Mundo!
===========

## Descripción:

<p align="justify">
	Es un hola mundo en Fortran 90 en Mac.
</p>


## Técnico:

* macOS Sierra 10.12.5

<p align="justify">
	Si eres desarrollador seguramente ya tendrás instalado Xcode y las herramientas necesarias para compilar desde linea de comandos, las cuales incluyen una <b>adaptación</b> de las <i>GNU Compilar</i> para compilar C o Fortran desde la terminal. 

Sino bastara instalar:
</p>

* Xcode desde App Store.

<p align="justify">
Me refiero a una adaptación de las <i>GNU Compiler</i> porque Apple utiliza <a href="http://llvm.org/"><i>LLVM Compilar Infrastructure</i></a> pero eso no nos debe preocupar pues para el usuario es transparente podemos teclear desde la terminal <b>gcc</b> o <b>gfortran</b> sin problema.
</p>
